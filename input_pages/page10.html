Introduction to Wikipedia

Wikipedia is made by people like you.

This page takes you through a set of tutorials aimed at complete newcomers who wish to contribute. It covers all the basics, and each tutorial takes only a few minutes, so you can become a proficient Wikipedian quickly.

Get started
Policies and Guidelines

The wiki markup source editor shows the underlying page source code. It works like a plain text file, indicating links and other items using simple code like this: [[Earth]].
